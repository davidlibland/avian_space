# Music provenance

Every file in this directory is created for this game — no third-party
music. Two generations of files share the same names:

1. **Placeholders (current)** — synthesized ambient loops from
   `scripts/synth_music_placeholders.py` (numpy, in-house). No external
   rights involved.
2. **Finished tracks** — composed as chord charts/briefs
   (`docs/biab_briefs/`) and rendered by the developer in Band-in-a-Box.
   PG Music's license grants royalty-free commercial use of songs
   created with Band-in-a-Box and its RealTracks; the rendered audio is
   an original work of the developer. (The RealTracks source material
   itself is never redistributed — only finished renders ship.)

When a finished track replaces a placeholder, note it here. For
rendered slots the synth pad is KEPT as `<key>_pad.ogg` — the music
director alternates between song and pad in-game.
