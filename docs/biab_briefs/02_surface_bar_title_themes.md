# BIAB brief pack 2 — surface biomes, the bar, and the title (7 tracks)

Same workflow as pack 1 (see `01_faction_space_themes.md`): preferred route is opening the song's `musicxml/` lead sheet (title/key/tempo/chords all import); text-paste fallback: **File | New** → set title/key/tempo by hand from the
table → copy the chart (below, or the matching file in `paste/`) →
**Edit | Paste Special – from Clipboard text to Song(s)** → pick a
style → **3 choruses, no generated ending** → Generate → **export WAV
44.1 kHz** named as the "File" row. Every form is 32 bars of 4/4 (the
loop processor assumes 128 beats — please keep 4/4 styles).

These replace the synth placeholders in `assets/music/`; like the
faction themes, the pads will be kept as alternating variants.
