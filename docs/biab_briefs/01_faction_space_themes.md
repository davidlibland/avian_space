# BIAB brief pack 1 — faction space themes (7 tracks)

How to use each brief (once per track, ~10 min):

1. Open Band-in-a-Box, **File | New**.
2. Copy the chord block below and press **Ctrl+Shift+V** (Edit | Paste
   Special – from Clipboard text to Song). Title/key/tempo come along.
3. Open the **StylePicker** and pick a style per the "Style" line — the
   named styles are suggestions; anything close from the same genre
   filter works. Trust your ear over my style pick.
4. Set **3 choruses**, and turn OFF the 2-bar ending (Song Settings →
   uncheck "Generate 2 bar ending") so the render doesn't resolve/decay.
5. **Generate**, listen, regenerate or swap RealTracks until it feels
   right. Small edits welcome — the chart is a starting point.
6. **File | Export | Save as WAV** (44.1 kHz). Name it exactly the
   target filename and drop it anywhere in the repo (or a folder you
   tell me about) — I trim the middle chorus into a seamless loop,
   normalize, encode ogg, and swap it in.

Every form is 32 bars so the loop point lands on a clean phrase.
Renders replace `assets/music/<name>.ogg` (placeholders are in there
now — worth a listen for the vibe each slot is aiming at).
